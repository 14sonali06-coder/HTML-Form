# CodeXcelerate Internship - HTML Form Assignment


##  About the Project
This project is created as part of my internship at **Codexcelerate**.  
It is a simple HTML form that collects user details such as Name, Email, Phone, and Gender.


##  Features
- Structured HTML form with proper labels
- Input validation using HTML attributes 
- Gender selection using radio buttons


##  Form Fields
- Name 
- Email 
- Phone 
- Gender 
- Submit Button
